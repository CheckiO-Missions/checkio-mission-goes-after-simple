Ok, womit soll ich anfangen?

Für diese spezielle Lösung verwenden wir die Suche nach dem Anfangsindex der Teilzeichenkette in der Zeichenkette. Dazu können Sie eine von zwei String-Methoden verwenden: str.find() oder str.index(). Diese Methoden funktionieren auch mit mehrstelligen Teilstrings. Lassen Sie uns ihre Ergebnisse vergleichen: 

line = "table"
# find
print(line.find("b"))
print(line.find("le"))
print(line.find("x"))

Ausgabe:

2
3
-1        
            
# index
print(line.index("b"))
print(line.index("le"))
print(line.index("x"))

Ausgabe:

2
3
ValueError: substring not found

Verwenden wir die erste, die -1 zurückgibt, wenn keine Teilzeichenkette in der Zeichenkette vorhanden ist. Das macht Ihre Lösung einfacher und kürzer. 

word.find(...) # returns starting index of substr (if present) or -1

Ich brauche Hilfe bei dem Argument der find()-Methode.

Das Zeichen second muss direkt nach first stehen. Das bedeutet, dass wir nach der Teilzeichenkette first + second suchen.

In der Zeichenkette "table" folgt zum Beispiel "b" direkt auf "a" - dazwischen befindet sich kein Zeichen. Da find() ein mehrstelliges Argument annehmen kann, können wir sagen, dass wir nach der Teilzeichenkette "a" + "b" -> "ab" in der Zeichenkette "table" suchen. 

word.find(first + second)

Diese Methode gibt eine Zahl zurück (Index oder -1). Wie erhält man bool aus ihr?

Ihre Funktion sollte True zurückgeben, wenn die Teilzeichenkette in word vorhanden ist. Das bedeutet, dass der Startindex (Ergebnis der Methode find ) nicht gleich -1 sein darf (größer, da er nicht kleiner sein kann). Lassen Sie uns die beiden vergleichen und so einen booleschen Wert erhalten. 

-1!= word.find(first + second)
# or
-1 < word.find(first + second)

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur Ihren Code in einen Funktionsausdruck einpacken und das Ergebnis des Vergleichs zurückgeben.

Hier ist die gesamte Lösung:

def goes_after(word: str, first: str, second: str) -> bool:

    return -1 < word.find(first + second)

Sie können die Anwesenheit auch mit dem Operator in überprüfen. 

def goes_after(word: str, first: str, second: str) -> bool:

    return first + second in word

Die andere Methode index() wird hier verwendet, die einen Fehler (ValueError) auslöst, wenn eine Teilzeichenkette fehlt. 

def goes_after(word: str, first: str, second: str) -> bool:

    try:
        return word.index(first) - word.index(second) == -1
    except:
        return False
