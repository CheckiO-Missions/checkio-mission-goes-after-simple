W podanym ciągu należy sprawdzić, czy jeden symbol występuje bezpośrednio po drugim. Jeśli tak - proszę zwrócić {% if is_js %}true{% else %}True{% endif %}, w przeciwnym razie - {% if is_js %}false{% else %}False{% endif %}.

Jeśli jeden z symboli nie znajduje się w podanym słowie - funkcja powinna zwrócić {% if is_js %}false{% else %}False{% endif %}. Jeśli dwa szukane symbole są takie same - funkcja powinna zwrócić {% if is_js %}false {% else %} False {% endif %}.

Wejście: Trzy argumenty. Pierwszy to podany ciąg znaków {% if is_js %}(string){% else %}(str){% endif %}, drugi to symbol {% if is_js %}(string){% else %}(str){% endif %} który powinien pojawić się jako pierwszy, a trzeci to symbol {% if is_js %}(string){% else %}(str){% endif %} który powinien pojawić się po pierwszym.

Wyjście: Wartość logiczna {% if is_js %}(boolean){% else %}(bool){% endif %}.
