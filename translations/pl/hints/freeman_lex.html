Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

ponieważ wszystkie znaki na wejściu word są unikalne, musi Pan tylko sprawdzić, czy podciąg ze znaków first i second znajduje się w word.

Ok, od czego powinienem zacząć?

W tym konkretnym rozwiązaniu używamy znalezienia indeksu początkowego podciągu w łańcuchu. Można to zrobić za pomocą jednej z dwóch metod łańcuchowych: str.find() lub str.index(). Metody te działają również z podciągami wieloznakowymi. Proszę porównać ich wyniki: 

line = "table"
# find
print(line.find("b"))
print(line.find("le"))
print(line.find("x"))

Wyjście:

2
3
-1        
            
# index
print(line.index("b"))
print(line.index("le"))
print(line.index("x"))

Wyjście:

2
3
ValueError: substring not found

Użyjmy pierwszego z nich, który zwraca -1 w przypadku braku podciągu w ciągu. Dzięki temu Państwa rozwiązanie będzie prostsze i krótsze. 

word.find(...) # returns starting index of substr (if present) or -1

Potrzebuję pomocy z argumentem metody find().

Znak second musi znajdować się zaraz po first. Oznacza to, że szukamy podciągu first + second.

Na przykład w ciągu "table", "b" znajduje się zaraz po "a" - nie ma między nimi żadnego znaku. Ponieważ find() może przyjmować argumenty wieloznakowe, możemy powiedzieć, że szukamy podciągu "a" + "b" -> "ab" w ciągu "table". 

word.find(first + second)

Metoda ta zwraca liczbę (indeks lub -1). Jak uzyskać z niej wartość bool?

Państwa funkcja powinna zwrócić True, jeśli podciąg jest obecny w word. Oznacza to, że indeks początkowy (wynik metody find ) nie musi być równy -1 (większy, ponieważ nie może być mniejszy). Porównajmy je i uzyskajmy w ten sposób wartość logiczną. 

-1!= word.find(first + second)
# or
-1 < word.find(first + second)

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy zawinąć kod w wyrażenie funkcyjne i zwrócić wynik porównania.

Oto całe rozwiązanie:

def goes_after(word: str, first: str, second: str) -> bool:

    return -1 < word.find(first + second)

Mogą Państwo również sprawdzić obecność za pomocą operatora in. 

def goes_after(word: str, first: str, second: str) -> bool:

    return first + second in word

W tym przypadku używana jest inna metoda index(), która zgłasza błąd (ValueError) w przypadku braku podciągu. 

def goes_after(word: str, first: str, second: str) -> bool:

    try:
        return word.index(first) - word.index(second) == -1
    except:
        return False
